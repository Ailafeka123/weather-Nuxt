# 氣象預告

## 說明

使用政府提供的開放API配合Nuxt去製作一個氣象查詢網站

目前擁有每個地區的36小時內氣象預告

使用Nuxt的sever製作api接口

使用plugins製作共用資訊且用router進行補助


## 進度

1. 待index畫面修正

2. 氣象預告的7日與3日預告架構

3. 目前思考要怎麼顯示，預計3日會出現溫度折線圖配合每一時間的最高最低溫、濕度、天氣，7日預計簡略成 標題為一周日期， 格子為每個地區(該縣市的鄉區)的當日最高最低溫，天氣(取最不好的)

4. 7日與3日預告的api製作大致完成，待製作對應的json好讓api可以呼叫到氣象開源API
